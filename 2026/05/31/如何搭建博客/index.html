为什么不用服务器？

很多人第一次搭博客，第一反应都是：

  • 买云服务器
  • 装 WordPress
  • 配数据库
  • 配 HTTPS

但实际上：

Hexo 本质上是：

你平时写的 Markdown：

最终都会在本地生成 HTML 文件。

而 GitHub Pages 会帮你直接托管这些静态页面。

也就是说：

整个过程不需要后端。

所以：

  • 不需要运维
  • 不需要防攻击
  • 不需要数据库
  • 甚至不需要服务器

这一点真的舒服很多。
